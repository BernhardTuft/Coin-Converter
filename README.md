# Coin-Converter
The Coin Converter is a C# program designed to simplify currency handling by converting any whole number into the fewest coins possible. Whether you're dealing with dollars, euros, or any other currency, this handy tool ensures you always have the optimal combination of coins.

Features
Convert any whole number into the fewest coins possible.
Utilizes gold coins (value: 10), silver coins (value: 5), and bronze coins (value: 1).
User-friendly interface for easy input and output.

Getting Started
Prerequisites
.NET SDK 
Installation
Clone the repository to your local machine:
git clone https://github.com/BernhardTuft/Coin-Converter.git
Navigate to the project directory:

cd coin-converter
Build the project:
dotnet build

Usage
Run the program:

dotnet run
Enter the amount you wish to convert when prompted.

The program will display the optimal combination of gold, silver, and bronze coins to represent the given amount.
